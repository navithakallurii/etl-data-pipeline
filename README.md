# etl-data-pipeline
# ETL Data Pipeline (Python + SQL)

## Overview
This project demonstrates a simple ETL (Extract, Transform, Load) pipeline.

## Steps
- Extract data from a source
- Transform data (remove null values and duplicates)
- Load cleaned data into a database
- Validate stored data using SQL queries

## Tech Stack
- Python (Pandas)
- SQLite (SQL)

## Use Case
This simulates real-world data preparation for analytics and machine learning pipelines, where data quality and consistency are critical.
